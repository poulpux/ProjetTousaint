Salut Tristan, voici le jeu où tu vas incarner SoftLizard, agent secret infiltrant la prestigieuse Ecole de Game Academy.

Le jeu se joue sur Mobile, ducoup je tu laisse télécharger la build avec l'éditeur. 
Sinon c'est un jeu vachement sympa qui se fini en 3 minutes. Il est composé d'un menu, 2 salles de tuto et 3 salles de jeu, avant de finir sur l'écran de fin.

Le tuto est assez simple alors je te laisse découvrir.

Mon seul regrès est d'avoir decouvert que trop tard que les PlayersPrefs ne marche pas sur mobile et donc il n'y aura pas de meilleur temps sur la version téléphone.

Sur ce j'espère que tu passeras un bon moment.
